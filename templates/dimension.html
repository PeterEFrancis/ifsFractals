Dimension

Notations:
  • $X$: a metric space with metric $d_X$
  • $S\subset X$
  • $B_\e(p)$: open ball of radius $\e$ centered at point $p$
  • $N_\e(S)$: the number of boxes of side length $\e$ required to cover $S$
  • $M_C(p)$: (given a cover $C$ of $S$ and $p\in S$) $M_C(p)$ is the cardinality of the set $\{U\in C\mid p\in U\}$.
Hausdorf Dimension
(See Wikipedia)
For $q\geq 0$, the $q$-dimensional Hausdorf measure of $S$ is $$ \mathcal{H}^{q}(S)=\lim_{\e \to 0} \inf \left\{\sum_{i} r_{i}^{q}\mid \text { there is a cover of } S \text { by balls with radii } 0 < r_{i} < \e\right\}.$$ The Hausdorff dimension of $S$ is defined by $$\dim_{\mathrm{H}}(S)=\inf \left\{q \geq 0\mid \mathcal{H}^{d}(S)=0\right\}.$$ If we let $Q=\{q\geq 0 \mid \mathcal{H}^q(S)=\infty\}$, then $$\dim_{\mathrm{H}}(S)=\begin{cases} 0 & |Q|=0\\ \sup Q & \text{otherwise}.\end{cases}$$ Related to inductive dimension: if $S$ is non-empty, then $\dim_\text{Haus}(S) \geq \dim_\text{ind}(S)$. Moreover, $$\inf_{R\cong S} \dim_\text{Haus}(R) = \dim_\text{ind}(S).$$ If $I$ is countable and $S=\bigcup_{i\in I} S_i$, then $$\dim_\text{Haus}(S)=\sup_{i\in I}\dim_\text{Haus}(S_i).$$ If $X$ and $Y$ are non-empty metric spaces, then $$\dim_\text{Haus}(X\times Y)\geq \dim_\text{Haus}(X)+\dim_\text{Haus}(Y).$$
Minkowski-Bouligand (Box-Counting) Dimension
(See Wikipedia)
The Minkowskiâ€“Bouligand (box-counting) dimension of $S$ is defined to be the limit $$\dim_\text{box}(S)=\lim_{\e\to\infty}\frac{\log N_\e(S)}{\log(1/\e)}.$$ If the above limit does not exist, one may still take the limit superior and limit inferior, which respectively define the upper box dimension and lower box dimension, denoted $\dim_\text{upperbox}$ and $\dim_\text{lowerbox}$. Related to Hausdorff: $\dim_\text{Haus}\leq \dim_\text{lowerbox}\leq \dim_\text{upperbox}$. It is possible that both inequalities are strict.
Menger-Urysohn (Inductive) Dimension
(From Hurewicz and Wallman)
Assume that $S$ is seperable. We define inductive dimension at a point $p$ of $S$, $\dim^p_\text{ind}(S)$, and then

inductive dimension

of $S$, $\dim_\text{ind}(S)$.
  • If for all $U$ containing $p$, there is some $V$ with $p\in V\subset U$ and $\partial V = \emptyset$, then $\dim^p_\text{ind}(S)=0$.
  • If $S$ is nonempty and has dimension 0 at each of its points, $\dim_\text{ind}(S)=0$.
  • Only the empty set has dimension $-1$.
  • For $p\in S$, if $\dim^p_\text{ind}(S)\leq n$ and it is not true that $\dim^p_\text{ind}(S)\leq n-1$, then $\dim^p_\text{ind}(S)=n$.
  • If for all $p\in S$, $\dim^p_\text{ind}(S)\leq n$, then $\dim_\text{ind}(S)\leq n$.
  • If $\dim_\text{ind}(S)\leq n$ and $\dim_\text{ind}(S)\leq n - 1$ is false, then $\dim_\text{ind}(S)= n$.
  • If $\dim_\text{ind}(S)\leq n$ is false for each $n\in\N$, then $\dim_\text{ind}(S)=\infty$.
Lebesgue Covering Dimension
Suppose that $C$ is a cover of $S$. The order of $C$ (if it exists) is the smallest $n\in\N$ for which all points of $S$ lie in at most $n$ sets in $C$. $$\Ord_S(C)=\min\{n\in\N\mid \forall p\in S, M_C(p)\leq n\}.$$
